How to Launch Cookie Clicker?

1. Download the cookie clicker file with MultiLauncher.


2. Find the ZIP folder in the file browser.


3. Right click the ZIP file and select EXTRACT ALL.


4. Open the UNZIPPED version of the file.


5. Locate the INDEX.HTML file inside the folder and launch it.


6. COOKIE CLICKER will launch! Bookmark it with google to return easily.
